Production API: [https://hunty-bot.vercel.app/api/messages](https://hunty-bot.vercel.app/api/messages)

Chat with Telegram Bot: [@Hunty23Bot](https://t.me/Hunty23Bot)

You can chat with bot about the colombian provinces, for example if you want know about Antioquia, Atlantico, Bogot√°.

Special thanks to [API-Colombia](https://api-colombia.com/)


Features:

- Environment, NodeJS, NextJS Setup
- MongoDB & Prisma connect, Database creation
- API and Controllers creation
- Telegraf Lib

### Prerequisites

**Node version 18.x**

### Cloning the repository

```shell
git clone https://github.com/djnet00/huntyBot.git
```

### Install packages

```shell
npm i
```

### Setup .env file


```js
BOT_TOKEN=
DATABASE_URL=
```

### Start the app

```shell
npm run dev
```

## Available commands

Running commands with npm `npm run [command]`

| command         | description                              |
| :-------------- | :--------------------------------------- |
| `dev`           | Starts a development instance of the app |
